# 1 Introduction

- 1.1 Document Purpose and Scope This document defines test cases specific to the Opal SSC 1.00, 2.00, 2.01 and 2.02; Opalite SSC 1.00; Pyrite SSC
- 1.2 Scope Not every feature or capability within those specifications is included in this document for testing. The test cases are driven by baseline capabilities of the SSC specifications, and by SD responses that can be verified by functional testing and are representative of expected use cases. The test cases do not include any compatibility testing between SSC versions.
- 1.3 Intended Audience The intended audience for this document is SD manufacturers and software developers that may wish to tie SDs into trusted platforms, as well as manufacturers and developers of other components that intend to bind to trusted SDs.
