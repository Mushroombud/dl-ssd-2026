# 1.3.7 Bit Conventions

- 1.3.7 Bit Conventions Name (n:m), where n is greater than m, denotes a set of bits (e.g., Feature (7:0)).
- 1.3.8 Number Range Conventions p..q, where p is less than q, represents a range of numbers (e.g., words 100..103 represents words 100, 101, 102, and 103).
- 1.3.9 Specify and Indicate Conventions In a given message, command, or other information exchange between a requestor (e.g., a host) and a responder (e.g., a Storage Device):
- a) the word ‘specifies’ means that the requestor provides information in the request; and
- b) the word ‘indicates’ means that the responder provides information in the response.
