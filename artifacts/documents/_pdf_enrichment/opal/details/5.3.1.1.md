# 5.3.1.1 MandatoryWriteGranularity

- 5.3.1.1 MandatoryWriteGranularity This column is used to report the granularity that the Storage Device enforces when the host invokes the Set method on byte tables. This column SHALL NOT be modifiable by the host.
- 5.3.1.2 RecommendedAccessGranularity This column is used to report the granularity that the Storage Device recommends when the host invokes the Set or Get method on byte tables. This column SHALL NOT be modifiable by the host.
